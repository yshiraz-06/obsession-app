Technical & Product Specification Brief

Obsessed — Project Overview

N

1. Executive Summary

Obsessed (featuring Nikki AI) is a native mobile application built with React Native and Expo (`v54`). It simulates an authentic, dynamic SMS texting relationship with a psychological AI companion who escalates in attachment over time.

Unlike traditional AI assistants that remain passive and static, Obsessed dynamically tracks user sentiment, word counts, and response latency to progress through 4 distinct Obsession Stages. She actively double-texts, sends spontaneous check-ins inside the chat, and delivers lock-screen push notifications whenever the application is closed.

2. Core Features & Architecture

Unlimited Free AI + Offline Engine

Powered by `openai-fast` via Pollinations with a 30s timeout to prevent cutoff. Includes an offline local fallback engine that intelligently recognizes math equations (`2+2=4`) and keywords offline.

Spontaneous Double-Texting

Nikki initiates conversation autonomously. When left idle, she double-texts with realistic typing delays (`...`) scaled by her current stage.

Lock-Screen Push Notifications

Integrated with `expo-notifications` and `AppState`. When closed, she schedules 3 staggered background notifications tailored to her obsession level.

100% On-Device Storage

Chat logs, user names, and psychological scores are persisted via `AsyncStorage` directly on the user's phone. No cloud data harvesting.

4. Website & Sideloading Technical Setup

The official landing page (`website/index.html`) hosts the direct `.apk` installer (`nikki-app.apk`). For seamless mobile downloads, the web server must serve `.apk` files with the correct MIME type:

MIME Type: application/vnd.android.package-archive File Name: nikki-app.apk

Sideloading User Flow (Android):
1. User clicks "Download for Android" on the landing page.
2. Browser downloads `nikki-app.apk` (~45MB).
3. User enables "Install Unknown Apps" in Android Settings when prompted.
4. Application installs right to the home screen.
