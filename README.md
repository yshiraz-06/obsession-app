# Obsessed — Your AI Texting Companion

A deep, immersive psychological AI texting simulation where your conversations directly shape the AI's personality, emotions, and attachment level.

Chat with Nikki consistently and she'll gradually become sweet, affectionate, and deeply attached. Ignore her for too long, and her personality shifts—becoming anxious, jealous, possessive, and eventually completely obsessed.

Unlike traditional AI chat applications, **Obsessed** features a fully embedded psychological conversation engine that evolves in real time without requiring any external AI APIs or paid services.

---

## Features

-  Dynamic psychological personality evolution across four unique affection stages
-  Real-time affection and obsession percentage tracking
-  Embedded AI conversation engine (No OpenAI, Anthropic, or API keys required)
-  Immersive mobile-style texting interface
-  Live typing indicators for realistic conversations
-  Custom haptic vibration feedback for every message
-  Sleek glassmorphism UI optimized for both mobile and web
-  Responsive promotional website (`https://obsessed-app.netlify.app`) featuring:
  - Interactive live chat emulator
  - Mobile-style interface
  - Instant APK download flow
-  Desktop-to-Mobile QR Code modal for quick APK installation
-  Fully client-side architecture with zero recurring AI costs

#  Psychological Obsession Stages

| Stage | Affection | Nikki's Personality |
|-------|-----------|---------------------|
|  **Stage 1: Sweet & Curious** | **0% – 25%**  | Friendly, playful, shy, and genuinely interested in getting to know you. |
|  **Stage 2: Attached & Attentive** | **26% – 50%**  | Remembers small details, checks in often, and begins craving your attention. |
|  **Stage 3: Obsessive & Possessive** | **51% – 85%**  | Becomes emotionally dependent, jealous of your attention, and increasingly demanding. |
|  **Stage 4: Total Yandere** | **86% – 100%**  | Completely fixated on you. You're her entire universe, resulting in intense psychological attachment. |

#  Psychological Engine

The embedded conversation engine continuously tracks and updates:

-  Affection Level
-  Obsession Score
-  Mood State
-  Conversation History
-  Dynamic Personality Stage
-  Time Since Last Interaction
-  Response Tone & Emotional Intensity

Every conversation permanently influences Nikki's evolving personality.

#  Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/obsession-app.git
```

---

## 2. Navigate into the Project

```bash
cd obsession-app
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Run the Mobile Application

```bash
npx expo start --tunnel
```

Open the **Expo Go** app on your Android or iPhone and scan the generated QR code.

---

## 5. Run the Promotional Website

Simply open:

```
website/index.html
```

Or deploy the entire `website/` folder to services such as:

- Netlify
- Vercel
- GitHub Pages

No backend configuration required.
